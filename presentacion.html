¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet una red mundial de computadoras conectadas entre sí. Mientras que WWW es la colección de información que es recolectada vía internet. (GeeksforGeeks, 2025)

¿Cuáles son las partes de un URL?

  • Protocolo: https
  • Dominio / Host: www.ejemplo.com
  • Puerto (opcional): 8080
  • Ruta: /ruta/pagina
  • Query string: ?x=1
  • Fragmento (opcional): #seccion

(Pukocz, 2022)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Obtener información (leer datos).
  • HEAD: Igual que GET pero sin el cuerpo (solo encabezados).
  • POST: Enviar datos al servidor (crear recursos).
  • PUT: Reemplazar un recurso completo.
  • PATCH: Modificar parcialmente un recurso.
  • DELETE: Eliminar un recurso.

(Métodos de Petición HTTP - HTTP | MDN, 2025)

POST, por ser más seguro al codificar el cuerpo que se esta enviando al servidor. (Ruseva, 2022)

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

No es responsabilidad del desarrollador por que indica que el navegador no puede recuperar la página solicitada que escribió en la barra de direcciones (acceso restringido, URL erroneá). (QuÉ Significa el Error 404 Not Found? 2025)

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Se produce cuando el servidor no reconoce o no admite la versión del protocolo HTTP especificada en la solicitud del cliente, es decir, el desarrollador tiene que entrar aquí para areglarlo. (GeeksforGeeks, 2025b)

Son etiquetas que se consideran obsoletas en las versiones modernas de HTML (especialmente HTML5) y cuyo uso no se recomienda.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML 5 soporta archivos multimedia, tiene etiquetas más semánticas, compatibilidad con varios navegadores, soporte para gráficos y juegos, formularios más avanzados, etc. (Barron, 2025)

¿Qué componentes de estructura y estilo tiene una tabla?

  • table: Define la tabla.
  • caption: Define el título de la tabla.
  • thead: Agrupa el encabezado de la tabla.
  • tbody: Agrupa el cuerpo de la tabla.
  • tfoot: Agrupa el pie de la tabla.
  • tr: Define una fila.
  • th: Define una celda de encabezado.
  • td: Define una celda de datos.
  • colgroup: Agrupa columnas para aplicar formato.
  • col: Define propiedades para una columna específica.

(W3Schools.com, s.f.)

¿Cuáles son los principales controles de una forma HTML5?

  • form: Define el formulario.
  • input: Campo de entrada de datos.
  • textarea: Área de texto multilínea.
  • select: Lista desplegable.
  • label: Etiqueta descriptiva para un control.
  • button: Botón para enviar o ejecutar acciones.

(Advanced Form Controls In HTML5, 2023)

Sobre el ciclo de vida y desarrollo de los sistemas de información: ¿Cuál es el ciclo de vida de los sistemas de información? ¿Cuál es el ciclo de desarrollo de sistemas de información?

Ciclo de vida de los sistemas de información

  • Análisis de viabilidad: Evalúa si el proyecto es técnica y económicamente posible.
  • Requisitos: Identifica las necesidades del usuario y del sistema.
  • Diseño: Define la estructura y funcionamiento del sistema.
  • Implementación: Desarrollo y construcción del sistema.
  • Pruebas: Verifica que el sistema funcione correctamente.
  • Despliegue: Puesta en marcha del sistema.
  • Mantenimiento: Corrección de errores y mejoras continuas.

(GeeksforGeeks, 2023)

Ciclo de desarrollo de los sistemas de información

  • Planificación: Define objetivos, alcance y recursos del proyecto.
  • Análisis: Recopila y estudia los requisitos del sistema.
  • Diseño: Estructura la arquitectura y componentes del sistema.
  • Desarrollo: Programación y construcción del sistema.
  • Pruebas: Verifica el correcto funcionamiento.
  • Implementación: Instalación y puesta en operación.
  • Mantenimiento: Soporte, correcciones y actualizaciones.

(Jackson et al., 2025)
