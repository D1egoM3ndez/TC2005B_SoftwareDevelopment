Preguntas de Javascript

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Semejanzas

  • Ambos usan sintaxis con llaves {}.
  • Ambos utilizan estructuras como if, for y while.
  • Ambos permiten programación orientada a objetos.
  • Ambos pueden trabajar con funciones y objetos.

Diferencias

  • Java es compilado; JavaScript es interpretado.
  • Java tiene tipado estático; JavaScript tiene tipado dinámico.
  • Java se ejecuta en la JVM; JavaScript se ejecuta en el navegador o en Node.js.
  • Java es más estricto en su estructura; JavaScript es más flexible.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getFullYear(): Devuelve el año.
  • getMonth(): Devuelve el mes (0 - 11).
  • getDate(): Devuelve el día del mes.
  • getDay(): Devuelve el día de la semana (0 - 6).
  • getHours(): Devuelve la hora.
  • getMinutes(): Devuelve los minutos.
  • getTime(): Devuelve los milisegundos desde el 1 de enero de 1970.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • push(): Agrega un elemento al final.
  • pop(): Elimina el último elemento.
  • shift(): Elimina el primer elemento.
  • unshift(): Agrega un elemento al inicio.
  • filter(): Filtra elementos según una condición.

Let

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • Pueden ser modificadas desde cualquier parte del programa.
  • Generan posibles conflictos de nombres.
  • Hacen el código menos seguro y más difícil de mantener.
  • Complican la detección de errores.
